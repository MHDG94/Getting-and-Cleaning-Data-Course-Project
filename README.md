# Getting-and-Cleaning-Data-Course-Project

This repository is for Getting and Cleaning Data Course Project. It includes three files: (1) Codebook.md, (2) run_analysis.R, and (3) FinalData.txt which will provide instructions and guidance on how to get as well as clean "Human Activity Recognition Using Smartphones" data set. 

Here is the dataset: https://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones

Files: 
(1) Codebook.md: This file shows both variables and data as well as any manipulations contributed by myself during getting and cleaning the data

(2) run_analysis.R: This file includes the R code that was used for getting and cleaning data 

(3) FinalData.txt: This file includes the cleaned data obtained via R code which is included within this repository
